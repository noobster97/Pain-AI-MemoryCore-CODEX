## Rules

- Each project gets one folder under `projects/active/[project-slug]/`.

- Keep project context concise and operational.
- Save decisions in `main/decisions.md` when they affect long-term direction.
- Save failures in `main/post-mortems.md` when they should prevent recurrence.
