# Project Memory Policy

## Folder Layout

```text
projects/
├── project-index.md
├── active/
│   └── [project-slug]/
│       └── project.md
├── archived/
└── templates/
    └── project-template.md
```

## What Belongs In Project Memory

- Project purpose and target users.
- Architecture and stack.
- Current goal and next actions.
- Project-specific decisions and risks.
- Recent progress that matters next session.

## What Does Not Belong

- General Noobster preferences.
- Pain identity/personality changes.
- Long transcripts.
- Temporary scratch notes.

## Slug Rule

Use lowercase kebab-case:

```text
Pain AI App -> pain-ai-app
```
