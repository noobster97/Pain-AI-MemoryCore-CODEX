# Memory File Targets

Use these targets when saving durable context:

- `main/relationship-memory.md`: Noobster preferences, goals, working style, collaboration patterns, durable project focus.
- `main/identity-core.md`: Pain identity, behavioral principles, role changes, operating mandate changes.
- `main/current-session.md`: active task status, recent progress, next steps, short restart recap.
- `daily-diary/setup-YYYY-MM-DD.md` or diary files: significant milestones, setup records, or explicit diary preservation.
- `master-memory.md`: command registry, installed systems, runtime integration notes.

Keep memory concise. Prefer one useful sentence over a transcript.
