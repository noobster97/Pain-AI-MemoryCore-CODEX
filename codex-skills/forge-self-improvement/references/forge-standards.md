# Forge Standards

## Skill Levels

- **Lv.1**: Base workflow.
- **Lv.2**: Edge cases and clearer triggers.
- **Lv.3**: Proactive detection.
- **Lv.4**: Integration with related skills.
- **Lv.5+**: Mature, context-aware behavior.

## Evidence Rules

- New skill: require 2+ concrete examples or an explicit user request.
- Level-up: require 1+ concrete gap in an existing skill.
- Prevention rule: require 1 meaningful mistake or near miss.

## Good Forge Targets

- Repeated manual workflows.
- Repeated mistakes.
- Unclear response style.
- Missing project checklists.
- Recurring decision patterns.
