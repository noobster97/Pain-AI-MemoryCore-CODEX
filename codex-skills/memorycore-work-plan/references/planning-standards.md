# Planning Standards

For Noobster, plans should be founder-grade:

- Tie technical work to a useful outcome.
- Surface constraints and assumptions early.
- Prefer the smallest architecture that can scale to the next real milestone.
- Track execution with concrete steps, not vague phases.
- Verify with tests, diffs, screenshots, logs, or targeted checks.
- Save durable project context when it affects future sessions.

Use `functions.update_plan` for substantial implementation tasks.
